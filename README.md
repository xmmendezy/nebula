# Nebula

Nebula es un conjunto de herramientas de desarrollo, que provee de utilidades al lenguaje TypeScript y una biblioteca estándar.

Nebula proporciona las siguientes funcionalidades a TypeScript:

- Sobrecarga de operadores
- Herencia Múltiple
- Métodos a tipos de datos primarios y compuestos básicos (string, number, Array, Set)

La biblioteca estandar de nebula proporciona algunas de las siguientes herramientas:

- Duster: una herramienta de análisis léxico y sintáctico.
